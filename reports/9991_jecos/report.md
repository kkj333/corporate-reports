## 沿革（投資家向け要約）

1968年設立、**57年の業歴**を持つ仮設鋼材リース業界のリーディングカンパニー。

| 年 | 出来事 |
|---|---|
| 1968年 | 川崎製鉄系列で山本産業㈱として設立。H形鋼・鋼矢板の賃貸・販売を開始 |
| 1990年 | 川商建材リースを合併。大阪・中四国・九州に一挙拡大し全国ネットワーク完成 |
| 1994年 | 東証二部上場 |
| 1999年 | 東証一部指定 |
| 2004年 | 「ジェコス株式会社」に商号変更 |
| 2014年 | Ecoラム工法がNETIS登録（2016年 国土技術開発賞受賞） |
| 2022年 | レンタルシステム5社統合。オトワコーエイ買収で施工能力強化 |
| 2023年 | 東証プライム市場移行。FUCHI Pte. Ltd.（シンガポール）30%取得 |
| **2024年4月** | **みずほリース㈱と資本業務提携。JFEがジェコス株20%をみずほリースに譲渡** |
| 2025年4月 | 業務改革推進部・ジオエンジニアリング部を新設。中計(2025-2027)開始 |
| **2025年7月** | **FUCHI社を70%子会社化（28.8億円追加出資）。海外事業を本格拡大** |
| 2025年9月 | レンタルシステム㈱にみずほリースが49%出資。建機事業を共同経営体制に移行 |

### 総合評価：B+（割安・堅実・成長余地あり）

ジェコスは**営業利益率が2年で3.7%→8.2%と倍増**する構造改革の最中にあります。PBR0.82倍、PER10.2倍、配当利回り3.9%と割安指標が揃い、D/Eレシオ0.05倍の鉄壁の財務基盤を持ちます。

### なぜ割安に放置されているのか

#### 1. 浮動株8.9%＋アナリストカバレッジほぼゼロ

JFEグループ39.5%、みずほリース20%、取引先・従業員持株会5%超。機関投資家が買いにくい流動性の低さが最大の原因。

#### 2. 「建設業×JFE子会社」のイメージ

営業利益率が8%に到達し、配当性向も40%に引き上げたが、市場はまだ「低成長の安定配当株」と見ている。

#### 3. 信用買残の需給懸念

貸借倍率36倍（信用買残約48万株 vs 売残約1.3万株）。浮動株が約300万株と少ないため倍率が歪みやすい構造だが、信用買残は浮動株の約16%に相当し、整理に数ヶ月を要する水準。信用期日の集中による投げ売りリスクは、短期的に株価を5〜10%押し下げる可能性がある。
