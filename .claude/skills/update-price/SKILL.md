# バリュエーション再計算

`$0` の企業レポートを、株価 `$1` 円で再計算する。

---

## 手順

### 1. 既存データ読み取り

`$0/report.md` から以下を取得:

- BPS、EPS（実績・予想）
- 発行済株式数（自己株除く）
- ネットキャッシュ
- 営業CF、FCF（実質FCF）
- 売上高
- 当期純利益
- 清算価値（1株あたり）
- DCF各シナリオの1株価値

### 2. 再計算対象

以下の指標を新しい株価で再計算する:

| 指標 | 計算式 |
|---|---|
| 時価総額 | 株価 × 発行済株式数 |
| PER（実績） | 株価 ÷ 実績EPS |
| PER（予想） | 株価 ÷ 予想EPS |
| PBR | 株価 ÷ BPS |
| PCR | 時価総額 ÷ 営業CF |
| PSR | 時価総額 ÷ 売上高 |
| 配当利回り | 予想年間配当 ÷ 株価 |
| PER×PBR | PER × PBR |
| EV/EBITDA | （時価総額 − ネットキャッシュ）÷ EBITDA |
| 清算価値ディスカウント | （清算価値 − 株価）÷ 清算価値 |
| DCF弱気 アップサイド | （DCF弱気1株価値 − 株価）÷ 株価 |
| DCFミドル アップサイド | （DCFミドル1株価値 − 株価）÷ 株価 |
| DCF強気 アップサイド | （DCF強気1株価値 − 株価）÷ 株価 |

### 3. 更新箇所

report.md の以下を更新:

- 冒頭の `**株価：{株価}円（{日付}）**`
- 「株価指標」セクションのテーブル全体
- 「企業価値」セクションのDCFシナリオ比較表のアップサイド列
- 「資産バリューチェック」の現在株価・ディスカウント率
- ネットキャッシュ vs 時価総額の時価総額
- 五段階評価のコメント（倍率が変わった場合）
- 総括の株価関連の記述

### 4. 確認

- 更新後、数値の整合性を確認する（PER×PBR、EV/EBITDA等が計算通りか）
