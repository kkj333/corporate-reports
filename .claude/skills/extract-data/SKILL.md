# 財務データ抽出

`$ARGUMENTS` で指定されたPDFファイルから財務データを抽出し、構造化されたテーブルとして出力する。

---

## 手順

### 1. PDFを読み取る

指定されたPDFファイルを Read ツールで読み取る。ページ数が多い場合は `pages` パラメータで分割して読む。

### 2. 抽出対象データ

以下の項目を優先的に抽出する（存在するもののみ）:

**有価証券報告書の場合:**

- 連結経営指標等の推移（5期分: 売上高、経常利益、当期純利益、包括利益、純資産、総資産、BPS、EPS、自己資本比率、ROE、PER、営業CF、投資CF、財務CF、現金同等物、従業員数）
- 連結BS（流動資産内訳、固定資産内訳、負債内訳、純資産）
- 連結PL（売上高、原価、売上総利益、販管費内訳、営業利益、営業外損益、経常利益、特別損益、法人税等、当期純利益）
- 連結CF計算書（営業CF内訳、投資CF内訳、財務CF内訳）
- セグメント情報（セグメント別売上・利益）
- 大株主の状況
- 配当の状況
- 役員報酬
- 事業等のリスク
- 経営上の重要な契約、後発事象、偶発債務
- 設備投資、減価償却費、研究開発費

**決算短信の場合:**

- 連結業績（売上高、営業利益、経常利益、当期純利益、EPS）
- 財政状態（総資産、純資産、自己資本比率、BPS）
- CF概況（営業CF、投資CF、財務CF）
- 配当の状況
- 業績予想
- セグメント情報

### 3. 出力形式

マークダウンテーブル形式で出力する。企業ディレクトリの `data/` フォルダに保存:

```
{証券コード}_{企業名}/
├── data/
│   ├── yuho_2024.md      # 有報データ
│   ├── tanshin_2025q4.md  # 決算短信データ
│   └── ...
```

各ファイルの先頭に出典情報を記載:

```
# データ抽出: {企業名}（{証券コード}）
**出典**: {ファイル名}
**抽出日**: {日付}
```

### 4. 注意点

- 数値は原文のまま抽出する（単位変換しない）。単位を必ず明記する
- △はマイナスを意味する。そのまま残す
- 注記・脚注で重要なもの（会計方針変更、組替等）は抽出する
- 読み取れない箇所は `[読取不可]` と明記する。推測で埋めない
- 抽出完了後、主要な数値（売上、利益、総資産等）の整合性を簡易チェックする
