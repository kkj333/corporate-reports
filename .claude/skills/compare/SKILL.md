# 企業比較

`$ARGUMENTS` で指定された企業ディレクトリのレポートを読み、横比較表を生成する。

---

## 手順

### 1. データ収集

指定された各企業の `report.md` から以下を抽出:

**基本情報:** 企業名、証券コード、市場、時価総額、従業員数

**バリュエーション:**
- PER（実績・予想）、PBR、PSR、PCR
- PER×PBR
- EV/EBITDA
- 配当利回り
- 清算価値ディスカウント

**収益性:**
- 売上高、営業利益、当期純利益
- 営業利益率、ROE、事業ROIC
- 売上CAGR（取得可能な期間）

**財務健全性:**
- 自己資本比率
- 有利子負債
- ネットキャッシュ
- NC / 時価総額 比率

**株主還元:**
- 配当性向
- 配当利回り
- 自社株買い実績

**成長性:**
- 売上成長率（直近期）
- 営業利益成長率（直近期）
- 海外売上比率（該当する場合）

### 2. 出力

ルートディレクトリに `compare.md` を生成する。

```
# 企業比較: {企業名1} vs {企業名2} vs ...
**作成日**: {日付}
```

### 3. 比較表フォーマット

各カテゴリごとにテーブルを作成:

```markdown
## バリュエーション比較
| 指標 | {企業A} | {企業B} | ... | 優位 |
|---|---|---|---|---|
| PER | 10.8倍 | 15.2倍 | ... | A |
```

- 「優位」列で、どちらが良い数値かを示す（A/B/ほぼ同等）
- 各カテゴリの最後に1〜2行のコメントを付ける

### 4. 総合評価

比較表の最後に総合評価セクションを付ける:

```markdown
## 総合評価

| 観点 | {企業A} | {企業B} |
|---|---|---|
| 割安度 | ★★★ | ★★☆ |
| 収益力 | ★★☆ | ★★★ |
| 財務安全性 | ★★★ | ★★☆ |
| 成長性 | ★★☆ | ★★★ |
| 株主還元 | ★☆☆ | ★★☆ |
```

### 5. ハイライト

比較表でも `hl-red` / `hl-amber` / `hl-green` を使う:
- 最も割安な指標 → `hl-green`
- 最も割高 or 要注意 → `hl-red`
- 同水準 → ハイライトなし

### 6. 注意点

- データの取得時点が異なる場合は明記する（決算期のずれ等）
- 業種が異なる企業同士の比較では、PSRやPERの単純比較に限界がある旨を注記する
- 各企業のレポートが存在しない場合は、その旨をユーザーに伝え、先に `/corporate-report` で作成を促す
